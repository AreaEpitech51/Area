# AREA

AREA est une plateforme d'automatisation de service, qui permet de connecter des API et mettre en places des actions automatiques.

## Stack

### Front-End

- [React](https://fr.react.dev/)
- [Next.js](https://nextjs.org/)
- [React Native avec Expo](https://expo.dev/)

### Backend

- [Next.js Route Handler](https://nextjs.org/docs/app/building-your-application/routing/route-handlers)
- [Lucia](https://lucia-auth.com/)
